## Overview
A 5-page Power BI report analyzing social media campaign performance across Instagram and Facebook.

## Tools & Technologies
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Dynamic Parameters & Slicers

## Key Metrics
| Metric | Value |
|--------|-------|
| Total Impressions | 339.81K |
| Total Engagements | 58.16K |
| CTR | 11.79% |
| Conversion Rate | 5.07% |
| Total Budget | 2.54M |

## Dashboard Features
- Side-by-side Instagram vs Facebook comparison
- Performance breakdown by ad type, gender, age group and country
- Dynamic parameter toggle for different KPIs
